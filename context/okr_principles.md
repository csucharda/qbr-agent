# OKR Principles (PBB)

## Core rules
- 3 objectives per area
- 3–5 key results per objective
- Objectives = clear, action-oriented, outcome-driven
- Key results = measurable, quantifiable, time-bound
- OKRs should link to upstream PBB priorities

## Writing guidance
- Focus on outcomes, not activity
- Use metrics wherever possible
- Make cause → effect explicit
- Align to strategic priorities (growth, engagement, efficiency)

## Expectations
- OKRs are aspirational (~70% achievement is healthy)
- Do not use OKRs for performance rating
- Keep them stable during the quarter
